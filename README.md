# <p align="center">Pyrus Player</p>

<p align="center"> A powerful, modern desktop music and video player built with Electron. </p>

<p align="center">  <b>Supported Audio Formats:</b> mp3, wav, ogg, flac, aac, m4a </p>
<p align="center">  <b>Supported Video Formats:</b> mp4, webm, mkv, mov, m4v </p>
